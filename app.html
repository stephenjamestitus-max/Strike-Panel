Cloud Sync — Supabase (Optional)
Access your data from any device. Your Supabase account — we never see your data.
Setup — takes 5 minutes
1Go to supabase.com → New Project (free tier is plenty).
2In the SQL Editor, run the schema below (copy → paste → Run).
3Go to Project Settings → API. Copy your Project URL and anon public key.
4Paste both below and hit Save & Connect. Done — all data syncs silently in the background.
SQL Schema — paste into Supabase SQL Editor
-- StrikePanel™ Cloud Sync Schema
-- Run this once in your Supabase SQL Editor

create table if not exists sp_sync (
  id         text primary key,
  key        text not null unique,
  value      text not null,
  updated_at timestamptz default now()
);

-- Allow anon read/write (RLS off for single-user setup)
alter table sp_sync enable row level security;
create policy "allow_all" on sp_sync
  for all using (true) with check (true);
